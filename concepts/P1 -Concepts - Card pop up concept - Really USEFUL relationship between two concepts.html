Conceptual Comparison

Logic 04

Occam’s Razor

The principle that entities should not be multiplied beyond necessity. When presented with competing hypotheses, the simplest explanation is usually the correct one.

Core Heuristic

"Select the solution with the fewest assumptions."

Application Context

  • • Scientific modeling
  • • Medical diagnostics (Zebra vs. Horse)
  • • Code refactoring & simplification
Behavioral 05

Hanlon’s Razor

The suggestion that we should never attribute to malice that which is adequately explained by stupidity or negligence.

Core Heuristic

"Presume incompetence over conspiracy."

Application Context

  • • Conflict resolution
  • • Organizational communication
  • • Interpersonal relationships

Relationship

Similarities

Both function as "razors"—philosophical tools used to "shave off" unlikely explanations or complex assumptions.

Differences

Occam focuses on logic & nature, while Hanlon focus on intent & agency within human systems.

Synthesis

"The simplest explanation for an error (Occam) is usually a mistake, not a plot (Hanlon)."
